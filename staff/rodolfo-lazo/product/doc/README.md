# MyPet

Bienvenido a MyPet, el espacio donde el cuidado de tus mascotas es prioridad. Registra fácilmente a tus compañeros peludos, lleva un control completo de su historial clínico y accede a toda su información en un solo lugar. Porque su bienestar merece atención organizada, accesible y segura.

## Tecnologías usadas

- HTML
- CSS
- JS

## Functional

### User

- Register
- Login
- Update credentials (username, password)
- Update profile (name,email, phone, license number,)

### Owner

- add pet
- remove pet
- modify pet
- list pets
- add comment log for pets
- remove comment log from pets
- modify comment log from pet
- list log for pets
- filter veterinarios

### Veterinary

- add action log for pet
- remove action log for pet
- modify action log for pet
